Cairn — pick an app icon

5 mockups by the design-polish teammate. Each shown full-size (256px) + at every size the app actually ships (.icns/.ico render). Toggle dark to see how each looks in macOS dark dock.

Background:
How to interpret: The 16px row matches what users see in their browser tab/favicon and macOS Spotlight. The 32px row is the system menubar. The 128px row is the dock at default magnification. Anything that turns into a blob at 16px is a dock-only icon — pick a different one if you want it to read as Cairn everywhere.
